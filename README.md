# SAP SE — Equity Research & Financial Modelling

## Project Overview
Full equity research analysis of SAP SE (SAP.DE), Europe's largest 
software company with €171.6B market cap.

## Tools & Methods
- **Python**: yfinance, pandas, matplotlib, numpy
- **Excel**: 8-tab financial model
- **Models applied**: Ratio Analysis | Pro Forma Forecasting | 
  Bond Valuation | Stock Valuation (DDM, P/E, P/B) | 
  CAPM | WACC | Capital Budgeting (NPV/IRR) | DCF Valuation

## Files
- `SAP_Analysis.ipynb` — Python notebook with all charts and calculations
- `SAP_Equity_Research.xlsx` — Full 8-tab Excel financial model
- `SAP_Equity_Research_Report.pdf` — 2-page professional report

## Key Findings
- Revenue grew 24% from 2022–2025, gross margin stable above 72%
- Beta: 0.909 — SAP less volatile than the DAX market index
- WACC: 7.30% | CAPM Required Return: 7.50%
- DCF Intrinsic Value: €209.74 | Average Fair Value: €172
- **Recommendation: BUY | 12-month Price Target: €172 | Upside: +23.5%**
